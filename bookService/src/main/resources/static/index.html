Добро пожаловать на сайт буккроссинга  !

 

Здесь вы можете оставить свою книгу, а затем забрать по необходимости.

Чтобы оставить свою книгу, необходимо указать следующую информацию в POST запросе в формате JSON 

по пути http://localhost:port/library/new.

1. Название книги (обязательно).

2. Имя автора (обязательно).

3. Жанр (не обязательно).

4. Год публикации (не обязательно).

После того как вы отдадите книгу, сервис вам сообщит, на какой полке она будет расположена.

Выдача книг осуществляется по номеру полки (id) с помощью GET запроса по пути http://localhost:port/library/id. Увидеть список доступных книг можно по http://localhost:port/library/all.

Убрать книгу из доступа можно при помощи DELETE запроса по пути http://localhost:port/library/id.
